# 原文快照说明

## 作用

`原文快照/` 保存少量核心李晨文章目录，包含 `index.html` 和文章图片资源。这样在没有完整文章库时，AI 仍能读取关键原文，而不是只看二手摘要。

公开仓库默认不携带这些完整 HTML 和图片。需要原文快照的用户，可以先从 `down.mptext.top` 自行导出李晨公众号文章到本地，再运行刷新脚本生成快照。

## 当前收录

快照清单由 `原文快照/manifest.json` 记录。主要覆盖：

- 家庭训练与分析人。
- 活色生香、吃喝玩乐、暧昧与亲密。
- AI 纠偏和专家式答案批评。
- 心性、思谋、手段、气度。

## 刷新方式

在有完整文章库的机器上运行：

```bash
python3 Lichen-LOVE-skill/scripts/refresh_article_snapshots.py --root . --skill-dir Lichen-LOVE-skill
```

其中 `--root` 指向从 `down.mptext.top` 导出的文章根目录。脚本会复制核心文章，并自动排除 `__analysis*`、`__pycache__` 等分析中间产物。

## 使用方式

当用户问题涉及某个专题时，优先读取对应快照：

- 不安全感、父母、家庭训练：读 `2014-03-08_女生如何克服不安全感_`、`2014-03-11_一种独生子女的教育方法`。
- 怯懦、攻击性、青年保守：读 `2014-01-21_主流教育的好处_敢斗敢胜_精神快乐_情感丰富`、`2014-04-04_为什么说90后是保守_单一_僵化的_`。
- 不会推进亲密：读 `2025-11-14_都没往留她过夜方面想____`。
- 暧昧与吃喝玩乐：读 `2026-06-01_吃喝玩乐_从早餐开始_`、`2026-06-05_暧昧是疲惫生活的解药`、`2026-06-09_暧昧的艺术性与美感`。
- AI 纠偏和具体人分析：读 `2026-06-20_研究亲密关系的顶级专家_真的会亲密吗_`、`2026-06-30_恋爱一年后突然失去了性欲`。

## 公开发布提醒

这些快照来自本地导出的文章。若将 skill 上传到公开 GitHub，请先确认原文和图片的版权、付费内容边界、转载许可。

如果无法确认，建议公开仓库只保留：

- `SKILL.md`
- `references/*.md`
- `references/文章资料库/*.md`
- `scripts/*.py`

并删除 `references/文章资料库/原文快照/`。

本仓库推荐公开发布方式就是：不上传 `原文快照/`，只保留本说明和刷新脚本。
